# Day 1 - Web đặt sân (khung tĩnh)

Mục tiêu hôm nay: hiểu cấu trúc HTML và dựng giao diện bằng Flexbox/Grid.

Cấu trúc:
- index.html : danh sách sân
- detail.html : lịch theo giờ
- css/style.css : style chung

Cách chạy:
1. Mở thư mục bằng VS Code
2. Cài extension "Live Server"
3. Right click index.html → Open with Live Server

Bài tập:
- Thêm 1 sân mới vào index.html
- Đổi màu header trong style.css
- Trong detail.html, thêm slot 22:00
